🔑 편집 시작하기

간편 비밀번호를 입력하면 바로 편집할 수 있습니다. 저장하면 GitHub에 자동 커밋되고, Vercel이 약 1분 안에 홈페이지에 반영합니다.

⚙️ 간편 비밀번호가 아직 작동하지 않는다면 (관리자 1회 설정)
  1. Vercel 로그인 → from-jemulpo-to-songdo 프로젝트 → Settings → Environment Variables
  2. ADMIN_PASSWORD = 원하는 비밀번호 / GITHUB_TOKEN = GitHub 토큰, 두 개 추가
  3. Deployments 탭 → 최신 배포의 ⋯ 메뉴 → Redeploy

GitHub 토큰 만들기: github.com/settings/personal-access-tokens/new → Only select repositories → from-Jemulpo-to-songdo → Permissions → Contents: Read and write → Generate

※ GitHub 토큰(github_pat_…)을 직접 붙여넣어도 똑같이 작동합니다. 비밀번호/토큰은 이 브라우저에만 저장됩니다.
